# matyr_chat
## set chat

```js copy

TriggerEvent("chatMessage", "name", "type", "text")
```
- name = player name or something 
- type = `normal` `warning` `error`
- text = custom

### custom type
``` js copy filename="@chat/html/css/style.css"
.type-name {
  background-color: rgba(0, 0, 0, 0.85);
}
```
