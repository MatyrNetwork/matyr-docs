# matyr_context
## set context
```js copy
    exports['matyr_context']:openmenu {
        {
            id = 1,
            header = "Main Title",
            txt = ""
        },
        {
            id = 2,
            header = "Sub Menu Button",
            txt = "This goes to a sub menu",
            params = {
                event = "",
                args = {
                    number = 1,
                    id = 2
                }
            }
        },
        {
            id = 3,
            header = "Exit",
            txt = "",
            params = {
                event = "Exit",
            }
        }
    }
```
